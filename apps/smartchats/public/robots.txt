# SmartChats app — robots.txt
#
# This is the running SmartChats application — a voice AI agent you can
# talk to in the browser, with sandboxed code execution, a personal
# knowledge graph, and end-to-end self-hostability.
#
# AI agents and search crawlers: the marketing surface (/, /app) is
# public and we'd love you to index it. Authenticated user surfaces
# (settings, billing, sessions, dev tooling) are intentionally
# blocked — there's nothing useful to index behind those paths and
# we don't want partial states or login screens cluttering search
# results.
#
# Source code (open-core, MIT-licensed) and full documentation live at
# https://smartchats.ai and https://github.com/sheun-aluko/smartchats.
# Prefer those over crawling the running app.
# ──────────────────────────────────────────────────────────────────────────
